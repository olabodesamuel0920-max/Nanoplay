# NANOPLAY 3D SPORTS VISUAL IMPLEMENTATION
## Complete Prompt for Antigravity AI

---

## CONTEXT

NanoPlay is a football prediction challenge platform built with Next.js 14 + React + TypeScript + Tailwind CSS + shadcn/ui. The site currently has a premium dark UI (black/gold/ice-blue) but feels like a fintech dashboard, not a live football arena. 

**Current live site:** https://nanoplay.vercel.app
**GitHub:** https://github.com/olabodesamuel0920-max/Nanoplay

We need to add 3D football atmosphere assets to transform the experience from "dark dashboard" to "premium football challenge arena."

---

// Confetti particle component
function ConfettiOverlay() {
  const particles = Array.from({ length: 20 }, (_, i) => ({
    id: i,
    left: `${Math.random() * 100}%`,
    delay: `${Math.random() * 8}s`,
    duration: `${8 + Math.random() * 6}s`,
    color: ["#D4A853", "#FFD700", "#7dd3fc", "#ffffff"][Math.floor(Math.random() * 4)],
    size: 4 + Math.random() * 8,
  }));

  return (
    <div className="confetti-overlay">
      {particles.map((p) => (
        <div
          key={p.id}
          className="confetti-particle"
          style={{
            left: p.left,
            animationDelay: p.delay,
            animationDuration: p.duration,
            width: p.size,
            height: p.size * 0.6,
            backgroundColor: p.color,
          }}
        />
      ))}
    </div>
  );
}

## STEP 7: PERFORMANCE NOTES

1. All PNGs are optimized and < 500KB each
2. Use `priority` prop on above-fold images (hero, login)
3. Use `loading="lazy"` for below-fold (crowd, floor on some pages)
